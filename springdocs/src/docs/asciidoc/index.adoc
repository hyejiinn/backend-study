== 개요

Spring REST Docs는 *테스트 코드가 통과해야만 문서가 생성*되는 방식이다. +
이 방식은 항상 최신 상태의 정확한 API 문서를 보장한다.

----
[동작 흐름]
 1. MockMvc 테스트 실행
 2. 요청/응답 캡처 → 스니펫(.adoc) 생성
 3. index.adoc에서 스니펫 include
 4. asciidoctor가 HTML로 변환
----
